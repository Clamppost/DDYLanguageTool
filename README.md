# DDYScratchView

* 跟随系统国际化

单纯根据系统语言，只需要把DDYLanguageTool文件夹拖入工程，然后在PCH或者AppDelegate 引入即可

* 应用内切换语言

这种方式可以在不重启App情况下国际化，但是局限在应用内控件，对displayName 和 系统弹窗(如权限弹窗)无效
